# 猫哥外贸独立站全栈前端项目 (maogeo-frontend)

基于 **Next.js 16 (App Router)** 打造的高性能外贸独立站前端。

## 🌟 核心特性
- **GEO + SEO 双引擎**：完美支持谷歌 SEO 与 AI 大模型 (ChatGPT, DeepSeek, Gemini) 生成式检索提炼 (RAG)。
- **极致性能**：PageSpeed 90+ 绿标秒开。
- **现代化视觉**：苹果极简风与蓝紫科技发光质感。

## 🚀 快速启动
```bash
npm run dev
```

访问 `http://localhost:3000` 即可预览。
